# Shake and Bake test

## Purpose
The shake and bake test is to be used as a stress test for the product, it will be used to test the product under extreme conditions, such as high temperature, vibration, and shock. The test will be used to identify any potential weaknesses in the product and to ensure that it can withstand the conditions it will be exposed to during its lifetime.

## method
The shake and bake test will be performed in a controlled environment that will meet or exceed the ProdSpec + SaftyMargin by "Validator set prosentage"

## Exspected data
the test must provide:
- ACC data in CSV format and/or abs max data
    - if abs max data only then max and mean jerk must be provided per frequency band tested
- Temperature data in CSV format and/or abs max,typ data
    - if a temprature map was used then the map must be provided with the data
- Proof of the test vidio or images

## pass/Fail criteria
The test will be considered a pass if the product meets or exceeds the ProdSpec + SaftyMargin for all tested conditions. If the product fails to meet the criteria, it will be considered a fail and will require further analysis and potential redesign.
